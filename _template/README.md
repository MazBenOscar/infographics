# Infographic Template

Folder layout for each infographic:

- index.html
  Web infographic (GitHub Pages entry point)
- sources.md
  Data sources and dates for all key metrics
- assets/
  Images, icons, and supporting media used by the HTML
- social/
  Social deliverables (IG carousel images, X thread text, LinkedIn copy)

Naming:
- Use kebab-case for folder names: example-idea-slug
