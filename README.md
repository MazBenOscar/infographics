# Infographics Collection

A collection of data visualizations and infographics created with [Infographica](https://github.com/openclaw/openclaw).

## Structure

Each infographic lives in its own folder at the repo root:

- `<slug>/index.html` — Web infographic (GitHub Pages entry point)
- `<slug>/sources.md` — Data sources and dates
- `<slug>/assets/` — Images and icons referenced by HTML
- `<slug>/social/` — Social deliverables (IG carousel, X thread, LinkedIn copy)

See `_template/README.md` for the standard layout.

### The $100K Trap: Credit Card Debt by Income
**[View Live →](https://mazbenoscar.github.io/infographics/credit-card-debt-100k-trap/)**

Households earning $100k-$150k now carry the highest credit card debt-to-income ratios. High earners are overextended, not just low-income.

**Key Findings:**
- $100k-$150k earners: 14.2% of income to credit card debt (highest ratio)
- Average balance for $100k+ households: $12,400
- Total U.S. credit card debt: $1.2 trillion

**Data Sources:** Federal Reserve Survey of Consumer Finances, TransUnion, Experian, NY Fed
